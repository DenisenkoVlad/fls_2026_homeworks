Властивості шрифтів

font-family: “main-font”, “fallback-font”, type; - родина шрифтів

body {font-family: "Arial", "Helvetica Neue", "Helvetica", sans-serif;}

Типи шрифтів:

  • serif - шрифт із зарубками (із засічками) - Times New Roman…
  • sans-serif - рублений (без засічок) - Arial, Roboto…
  • fantasy - декоративні - Classica Prestige, Papyrus, Historycal…
  • cursive - курсивні шрифти - Lucida Handwriting...
  • monospace - моноширинні - Courier New...

font-size - розмір шрифту

body {font-size: 30px;}

За замовченням 16px, якщо в браузері не вказано іншого

font-style - стиль шрифту

.title {font-style: italic;}

Стилі шрифтів:

  • normal (за замовченням) - звичайний (не змінений) стиль
  • italic - курсив
  • oblique - нахилені літери

font-weight - насиченість (товщину) шрифту (за замовченням 400 - regular)

Числове значення - словесне значення:

  • 100 - Thin (Hairline)
  • 200 - Extra Light (Ultra Light)
  • 300 - Light
  • 350 - Semi Light
  • 375 - Text
  • 400 - Regular (Normal / Book)
  • 500 - Medium
  • 600 - Semi Bold (Demi Bold)
  • 700 - Bold
  • 800 - Extra Bold (Ultra Bold)
  • 900 - Black (Heavy)
  • 950 - Extra Black (Ultra Black)

Властивості тексту

color - колір тексту (значенням може бути слово, код #, rgba - для прозорості, інші кодування…)

body {color: black;color: #000;color: rgba(255, 255, 255, 0.5); - прозорість 50%}

text-align- вирівнювання тексту

body {text-align: left; - за замовченнямtext-align: center;text-align: right;text-align: justify;}

Значення text-align:

  • left
  • center
  • right
  • justify - по ширині об’єкта (пробіли автоматичні, різні за розміром)

text-decoration - додає тексту певне оформлення

Значення text-decoration:

  • text-decoration - загальний запис для властивостей
  • text-decoration-line - встановлює тип декору
    • underline - лінія під текстом
    • overline - лінія над текстом
    • line-through - закреслює текст
    • none - (за замовченням) скасовує встановлений раніше параметр
  • text-decoration-color - змінює колір декору
  • text-decoration-style - змінює стиль декору
    • solid - суцільна лінія (за замовченням)
    • double - подвійна суцільна лінія
    • dotted - крапки
    • dashed пунктир
    • wavy хвиляста лінія
  • text-decoration-thickness - встановлює товщину лінії
  • text-underline-offset - встановлює відстань від тексту до лінії

text-shadow: гор_зміщення верт_зміщення розмиття колір; - додає тінь тексту, може бути мульти-тінь (значення через кому)

.text {text-shadow: 1px 1px 10px blue, -5px -5px 10px red;}

text-transform - регістр літер

.text {text-transform: uppercase;}

Значення text-transform:

  • capitalize - кожне слово починається з заглавної літери
  • lowercase - усі літери прописні (нижній регістр)
  • uppercase - усі літери заглавні (верхній регістр)
  • none (за замовченням) - відміняє ефект

line-height - міжрядеовий інтервал

.text {line-height: 1.25;}

Розрахунок - line-height / font-size (Плагін CSS Responsive Helper, автор Pavlo Krovitskyi)

Записувати слід в частинах (1.25), якщо писати у відсотках (125%), то рахуватиметься від font- size того елемента, для якого вказаний відсотковий line-height, тобто не буде наслідуватись для об’єктів, де line-height інший

letter-spacing - інтервал між літерами

.title {letter-spacing: 10px;}

word-spacing - інтервал між словами (розмір пробілів)

.title {word-spacing: 3px;}

white-space - керування поведінкою пробілів

.title {white-space: nowrap;} Lorem ipsum, dolor sit amet consectetur adipisicing elit. Ratione praesentium nostrum animi, totam doloribus voluptas sunt nobis, vero quam consequatur vel maxime. Veritatis consectetur, dolorem quasi consequuntur et nulla delectus?

text-indent - відступ першого рядка, абзац (червоний рядок)

.title {text-indent: 100px;}

text-wrap - рівномірно розташовує перенесений текст, добре працює в заголовках

.text {text-wrap: balance; }
